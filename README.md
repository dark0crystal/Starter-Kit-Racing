# Starter Kit Racing

A JavaScript/Three.js port of [Kenney's Starter Kit Racing](https://github.com/KenneyNL/Starter-Kit-Racing).

[Live demo](https://mrdoob.github.io/Starter-Kit-Racing/)

## How to Run

Clone the repository and serve it with any local HTTP server:

```bash
git clone https://github.com/mrdoob/Starter-Kit-Racing.git
cd Starter-Kit-Racing
npx serve .
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

Any static file server will work — for example `python -m http.server` or the Live Server extension in VS Code.
